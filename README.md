Configs
=======

Various .rc files

Set up
------

### Bash

    $ echo '. `pwd`/.bash_profile' >> ~/.bash_profile
    $ . ~/.bash_profile

### Git

    $ ln -sf `pwd`/.gitconfig ~/.gitconfig

### Vim

    $ ln -sf `pwd`/.vimrc ~/.vimrc

### Sublime Text 2

    $ ln -sfn `pwd`/sublime ~/Library/Application\ Support/Sublime\ Text\ 2/Packages/User

### Zsh

    $ ln -sfn `pwd`/zsh ~/.oh-my-zsh/custom
